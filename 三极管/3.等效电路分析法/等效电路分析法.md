# 等效电路分析法

## 直流

用大信号等效模型，算出$I_{CQ}$、$V_{CEQ}$（检查是否到饱和区）

$$I_B = \frac{V_{BB}-V_{BE(on)}}{R_B}\\I_C = \beta I_B \\V_{CE} = V_{CC}-I_CR_C$$

$R_D$可以忽略不计
